1. 개발환경으로는
window 10 / 이클립스를 사용했습니다.

os 운영체제로는
아마존 Ec2(가상컴퓨팅) 를 이용해서  RedHat(클라우드서비스)의  리눅스를 사용했습니다.


서버로는 apache 톰캣 9.0 버전을 사용했고

프레임워크(뼈대,틀 뜻) 
스프링부트 2.6.2 버전, 

자바 11버전을 사용했습니다.


템플릿 엔진으로는 타임리프를 사용했고, 


데이터베이스 관리 시스템(DBMS)으로는 
아마존 RDS 의 마리아DB 로 사용했습니다.


2. 보안설정은
Security config 클래스에서 접근 권한을 설정하고 관리를 할 수 있게 
설정하였습니다. 

일반 사용자는 hasRole 에 유저권한, 관리자는 hasRole에 어드민 권한을 주어
접근할 수 있도록 하였습니다.  

인증이 필요한 페이지는 인증을 할 수 있도록 로그인페이지로 이동처리를 했습니다.


3. 기능 구현으로는
회원가입 페이지에서 사용자가 입력한 이메일, 비밀번호는 정규표현식을 통해
알맞은 형식으로 입력될 수 있도록 자바스크립트로 표현하였습니다.


가입완료를 누르면 해당 정보는 DB에 저장되고, 로그인 페이지로 이동합니다.

4. 로그인 화면에서 존재하는 이메일인지 확인하고 존재한다면
입력한 비밀번호의 encode 된 값과 DB 암호화된 값을 비교하고 일치하면
정상 로그인 됩니다.

구글 아이디로 로그인 가능합니다.

5. 게시판은
비회원은 게시판 목록까지 조회가능하고 글쓰기, 상세페이지는
로그인을 해야 접근할 수 있도록 설정하였습니다.

controller 에서 글목록, 글쓰기, 수정, 삭제 매핑 설정후
서비스 임플에서 구체적인 기능들이 수행할 수 있도록 기능구현을 하였습니다.
